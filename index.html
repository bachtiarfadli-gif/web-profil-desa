Sistem Informasi Geografis Berbasis Web

Tentang Sistem

Website ini merupakan Sistem Informasi Geografis (SIG) berbasis web yang menyajikan data spasial wilayah desa dan kecamatan dalam bentuk peta interaktif.

Fitur Utama

  • Peta interaktif berbasis web
  • Visualisasi data wilayah desa dan kecamatan
  • Layer batas wilayah, jalan, bangunan, dan fasilitas umum
  • Kontrol layer dan zoom level
